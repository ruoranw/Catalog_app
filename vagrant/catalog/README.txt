About 

This is a catalog app that allows users to view, create, edit restaurants and menus. Users 
can login in with their google or facebook accounts.

How to run it

Download or clone the whole catalog folder and run it on your local machine. In terminal, 
cd your file directory and run python finalProject.py
Then type http://localhost:8101/ as address in your browser. You can use this app now!
